# Project Layout

一个 `flask` 项目可以单单由一个文件构成：

```python
from flask import Flask

app = Flask(__name__)


@app.route('/')
def hello():
    return 'Hello, World!'
```

`flask` 项目的 `.gitignore` 可以通过以下的方式编写：

```gitignore
venv/

*.pyc
__pycache__/

instance/

.pytest_cache/
.coverage
htmlcov/

dist/
build/
*.egg-info/
```
