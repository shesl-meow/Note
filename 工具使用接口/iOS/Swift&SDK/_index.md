# README

> 菜鸟教程：
>
> - https://www.runoob.com/swift/swift-tutorial.html

xcode 提供了 playground 功能学习 swift 的基础语法。

若要创建 OS X 的 playground，需要使用引入 Cocoa：

```swift
import Cocoa
```

若要创建 iOS 的 playground，则需要引入 UIKit：

```swift
import UIKit
```

输入输出可以用：

```swift
print("Hello World")

let world = readLine()
```
