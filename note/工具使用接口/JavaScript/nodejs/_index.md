# README

什么是 `node.js`?

- 免费的、跨平台的、以 javascript 作为语言的开源服务器环境。

`node.js` 可以做什么？

1. 生成一个动态的前端页面；
2. 可以操作服务器上的文件系统、数据库；
3. 可以接受处理表单信息。
